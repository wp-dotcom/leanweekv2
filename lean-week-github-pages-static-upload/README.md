# Lean Week — GitHub Pages Static Upload

This folder is already built and ready for GitHub Pages.

## Easiest setup

1. Create a new GitHub repository, for example `lean-week`.
2. Upload **all files and folders inside this folder** to the root of the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Set branch to `main` and folder to `/ (root)`.
6. Save.

Your app will be available at:

```text
https://YOUR-GITHUB-USERNAME.github.io/lean-week/
```

## iPhone use

Open the GitHub Pages URL in Safari, tap Share, then tap **Add to Home Screen**.

## Saved data

The app saves data locally on your iPhone/browser. Clearing Safari website data may remove saved recipes, photos, settings, grocery checks, and saved meal plans.
